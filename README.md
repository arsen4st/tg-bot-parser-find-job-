# 🚀 Rabota Parser — парсер вакансий из Telegram

Быстрый асинхронный парсер вакансий из Telegram-каналов. Сам следит за новыми постами, фильтрует по твоим требованиям и шлёт в личку бота с кнопкой «Перейти» на конкретный пост.

## ⚡ Особенности

- **Максимальная скорость**: asyncio + Telethon в режиме реального времени, без лишних запросов.
- **Фильтр по требованиям**: включи/исключи ключевые слова (`+python +удалённо -senior`).
- **Прямые ссылки**: каждая вакансия — кнопка, ведущая ровно в тот пост и тот канал.
- **Удобный UX**: инлайн-меню, reply-меню, добавление каналов по ссылке.
- **Production-ready**: rate limiting, проверка админа, глобальный обработчик ошибок.
- **Лёгкий**: SQLite, минимум зависимостей, работает на ноутбуке в фоне.

## 📦 Установка

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## ⚙️ Настройка

1. Скопируй `.env.example` → `.env`.
2. Получи **BOT_TOKEN** у [@BotFather](https://t.me/BotFather).
3. Получи **API_ID** и **API_HASH** на [my.telegram.org/apps](https://my.telegram.org/apps).
4. Узнай свой Telegram ID у [@userinfobot](https://t.me/userinfobot) и впиши в `ADMIN_ID`.
5. Подпишись в Telegram на каналы, которые хочешь парсить.

## 🚀 Запуск

### Обычный запуск

```bash
python main.py
```

Или просто запусти `start.bat` (с консолью) или `start_background.vbs` (фоном, без окна).

### Авторизация

При первом запуске бот запросит авторизацию Telegram-аккаунта (для чтения каналов). Отправь боту команду `/login` — он пришлёт QR-код. Отсканируй его:

```
Telegram → Настройки → Устройства → Подключить устройство → Сканировать QR
```

После успешной авторизации перезапусти `main.py`.

## 🤖 Команды бота

- `/start` — главное меню
- `/addchannel` — добавить канал (по ссылке или @username)
- `/removechannel <@channel>` — удалить канал
- `/channels` — список каналов
- `/setreq <+слово -слово>` — задать фильтр требований
- `/requirements` — посмотреть текущие требования
- `/latest [N]` — последние N вакансий (по умолчанию 10)
- `/pause` — приостановить рассылку
- `/resume` — возобновить рассылку
- `/status` — статус парсера
- `/login` — авторизация через QR-код

### Примеры фильтров

```
/setreq +python +удалённо -senior -опыт
/setreq +frontend +react +удаленка
/setreq +сборщик +москва
```

```
├── bot.py          # Инициализация бота
├── parser.py       # Telethon-клиент
├── database.py     # SQLite
├── config.py       # Конфигурация
└── main.py         # Точка входа
```

- Только `ADMIN_ID` может управлять ботом.
- Rate limiting защищает от случайного спама.
